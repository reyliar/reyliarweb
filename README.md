# reyliar.xyz

A static webmaster portfolio website for `reyliar`. The default language is English, with a Turkish language option built into the page.

## Editing

- Page structure and default English copy: `index.html`
- Visual design and responsive layout: `styles.css`
- Language switching and navigation state: `script.js`
- Hero image: `assets/webmaster-workspace-hero.png`

## Content Notes

The site is intentionally static and can be hosted on GitHub Pages, Cloudflare Pages, or any basic static host. The existing `CNAME` points to `reyliar.xyz`; no backend worker is required.

Update the portfolio examples, service copy, and contact links directly in `index.html` and mirror any translated text in `script.js`.
